## 📋 Descripción del Proyecto

GameNexus es una landing page moderna y responsiva para una plataforma de videojuegos. El proyecto fue desarrollado como parte del **Taller Práctico de Metodología BEM** del curso de Desarrollo de Aplicaciones Web, aplicando todos los conceptos de nomenclatura CSS escalable.

### 🎯 Objetivos del Taller

- Implementar la metodología **BEM (Block, Element, Modifier)** para CSS.
- Crear una landing page completa con **7 secciones obligatorias**.
- Organizar el código CSS por componentes independientes.
- Utilizar PHP para modularizar el HTML con `includes`.
- Documentar las decisiones de diseño BEM.
